    Rational number/Fraction

    A rational number is a number that can be expressed as the quotient or fraction p/q of two integers, a numerator p and a non-zero denominator q.

    addSubMulDiv

    Fraction operations: Addition, subtraction, multiplication, division and upend/reciprocal.

    Algorithm

    Complexity

    Sources

    • Rational number
    • Fraction

    Using

    import com.github.artemkorsakov.numbers.RationalNumber

val x = new RationalNumber(1, 3)
// x: RationalNumber = RationalNumber(x = 1, y = 3)
val y = new RationalNumber(5, 7)
// y: RationalNumber = RationalNumber(x = 5, y = 7)
x.add(y)
// res0: RationalNumber = RationalNumber(x = 22, y = 21)
x.sub(y)
// res1: RationalNumber = RationalNumber(x = -8, y = 21)
x.mul(y)
// res2: RationalNumber = RationalNumber(x = 5, y = 21)
x.div(y)
// res3: RationalNumber = RationalNumber(x = 7, y = 15)
x + y
// res4: RationalNumber = RationalNumber(x = 22, y = 21)
x - y
// res5: RationalNumber = RationalNumber(x = -8, y = 21)
x * y
// res6: RationalNumber = RationalNumber(x = 5, y = 21)
x / y
// res7: RationalNumber = RationalNumber(x = 7, y = 15)
x.upend
// res8: RationalNumber = RationalNumber(x = 3, y = 1)
y.upend
// res9: RationalNumber = RationalNumber(x = 7, y = 5)
