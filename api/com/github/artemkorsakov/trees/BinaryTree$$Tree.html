    A binary tree is either empty or it is composed of a root element and two successors, which are binary trees themselves.

    A binary tree is either empty or it is composed of a root element and two successors, which are binary trees themselves. An End is equivalent to an empty tree. A Branch has a value, and two descendant trees. A tree with only a root node would be Node('a').
