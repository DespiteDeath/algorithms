## Installation

algorithm is available on scala 2.11, 2.12, 2.13.

Add the following to your build.sbt
```scala
libraryDependencies ++= Seq(
  "com.github.artemkorsakov" %% "algorithm" % "0.0.1-SNAPSHOT")
```
